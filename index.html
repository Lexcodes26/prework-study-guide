HTML

  • Hypertext Markup Language (HTML) desribes the structure of a webpage
  • HTML consists of a series of elements
  • HTML tells the browser how to display content
  • The head element contains information about the webpage.
  • The body elemement represents the visible content shown to the user.
  • The opening tag is used to state where the element starts
  • Content (body) is information that the users see.
  • Closing tag is used with a forward slash before element

CSS

  • Cascading style Sheets (CSS) is used to for styling the webpage layout.
  • Here you change backgrounds, colors, layouts, spacing and animations
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside and element

Git

  • You access git using the comand line and git commands always start with git
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
